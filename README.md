## 一、這是什麼

「街知巷聞」是一個**真正會「做任務」的 AI 智能體**（人設：本地老街坊「阿濠」）。
你只要用一句自然語言講低需求，佢就會：

1. **規劃** —— 拆解你的日期、人數、興趣、預算、步行偏好；
2. **調用工具** —— 自動查天氣、搜景點、逐一核實開放時間、預測人流、計算步行路線、估算預算；
3. **多步執行 + 失敗恢復** —— 遇到「景點當日休息／預算超支／行得太遠」，會**自動改線、替換、縮減**；
4. **智能導流（核心創新）** —— 把你由逼爆的大三巴／議事亭，**分流到舊區老街與本地老字號**；
5. **輸出可驗證行程** —— 支援一日與 2–5 日多日行程，地圖路線、時間軸、人均預算、步行距離，每一項都列明、可核對。

整個過程在網頁上**實時流式展示**，評審可以一眼睇到智能體點樣「諗嘢、用工具、改線」。

## 四、如何運行（直接可運行）

### 方式 A：一鍵啟動
- **Windows**：雙擊 `run.bat`
- **macOS / Linux**：`bash run.sh`

腳本會自動安裝相依套件、啟動伺服器，並打開 <http://127.0.0.1:8000>。

### 方式 B：手動
```bash
pip install -r requirements.txt
cd backend
python -m uvicorn app:app --host 127.0.0.1 --port 8000
# 瀏覽器打開 http://127.0.0.1:8000
```

> **離線即可完整演示**：未設定 API Key 時，系統以「離線示範引擎」驅動，**同樣**會調用全部真實工具、用真實澳門數據完成規劃與失敗恢復。

### 接入真實 Qwen（阿里雲百煉）
把 `.env.example` 複製為 `.env`，填入大會提供的百煉代金券 Key：
```
DASHSCOPE_API_KEY=sk-xxxxxxxx
QWEN_MODEL=qwen-plus
```
重啟伺服器後，右上角徽章會由「離線示範引擎」變為「Qwen」，改由真實千問大模型透過 **OpenAI 相容接口** 驅動 ReAct 迴圈。

## 五、技術架構（對標 QwenPaw 五層架構）

```
瀏覽器 SPA（Leaflet 地圖 + SSE 實時智能體軌跡）
        │  Server-Sent Events
FastAPI 應用 (backend/app.py)          ← QwenPaw「Agent 運行時 / FastAPI app」
        │
ReAct 智能體 (backend/agent.py)        ← 規劃 / 工具調用 / 多步 / 失敗恢復
   ├─ Qwen 腦 (DashScope OpenAI 相容, function calling)
   └─ 離線腦（腳本化，調用同一套真實工具，保證可演示）
        │
工具集 (backend/tools.py)              ← 7 種工具，統一 schema（對標 QwenPaw Toolkit）
        │
知識庫 (backend/data/pois.json)        ← 70 個真實澳門景點（坐標/圖片來自維基、人手校正）
```

**七種工具**：`search_attractions`（景點檢索）、`get_weather`（天氣）、`check_opening`（開放時間核實）、`predict_crowd`（人流預測）、`find_local_gem`（舊區導流）、`compute_route`（步行路線）、`estimate_budget`（預算估算）。

## 六、資料來源與真實性

- **景點坐標 / 照片**：以程式抓取自 **Wikipedia / Wikimedia Commons**（公開授權），坐標用於真實的步行距離計算。抓取腳本見 `backend/data/scrape_*.py`。
- **開放時間 / 休息日 / 費用 / 人流特徵**：依公開資料人手整理（如鄭家大屋逢週三休、龍環葡韻逢週一休），用於開放核實與失敗恢復。
- **天氣 / 人流**：以季節與時段模型估算（可日後接入真實 API），介面已明確標示為估算值。

## 七、AI 倫理

- **透明**：行程由 AI 生成，介面明確標示；人流與天氣為估算值，請以現場為準。
- **準確**：景點開放時間、坐標等關鍵事實基於知識庫，模型不得杜撰；超出知識庫不亂作。
- **公平與在地關懷**：刻意把流量導向弱勢的舊區小店，促進社區可持續，而非加劇過度集中。
- **私隱**：不收集個人身分資料；無需登入即可使用。
